# app/ — frontend del Atlas México–EEUU

Este es el paquete Next.js del proyecto. Para contexto completo (qué es el proyecto, cómo correr el pipeline de datos, credenciales necesarias, cómo desplegar), ver el [README de la raíz del repo](../README.md).

## Comandos

```bash
npm install --legacy-peer-deps
npm run dev      # sincroniza /data y levanta el servidor de desarrollo
npm run build    # exportación estática (output: 'export')
npm run lint
```
